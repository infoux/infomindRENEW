inCMS

inCMS

전자정부프레임워크 기반의 안정적인 CMS 솔루션 입니다.
Open Source를 적극 활용하여 호환성 및 안정성을 극대화 하였습니다.

  • 반응형 웹
  • 시큐어코딩
  • 크로스브라우징
  • HTML5

운영환경 OS 및 특정 DB에 관계없이 구동

개발환경
Java Debeloment Kit 7.0, Servlet 8.0
지원 Client
JDK 7.0 이상 WAS (Apache Tomcat, Jboss application platform, Tmax JEUS, Oracle WebLogic Server)
DBMS
Oracle, Mysql, Tibero, Mssql

주요기능

  • 사이트관리
    • 멀티도메인 지원
    • 부서별 메인관리 (서브패스)
    • 페이지별 평가 기능
    • 부서별 그룹권한 관리
    • 접속 권한 관리 (IP, 블랙리스트 관리)
    • 금지어 설정
    • 위험알림 설정
  • 인증관리
    • 사용자 인증
    • SSO 지원
    • 소셜로그인 지원 (네이버 / 구글 / 페이스북 / 카카오톡)
  • 게시판
    • 무한 카테고리
    • 스킨관리
    • 갤러리 / 공지 / 동영상
    • 멀티업로드
  • 포털 연동
    • 웹 포털 크롤링 다음 / 네이버 블로그 API 연동
    • Facebook 게시물 연동
  • 전자정부 연동
    • 설문조사, 로그관리, 팝업 등
    • 메일전송
  • 스케쥴 관리
    • 자동 스케쥴 관리
